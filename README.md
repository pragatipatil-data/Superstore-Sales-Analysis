#  Superstore Sales Analysis

## Overview
Analysis of 9,994 sales records using SQL, Excel and Power BI.

## Tools Used
- PostgreSQL — Data extraction and analysis
- Excel — Pivot Tables and Dashboard
- Power BI — 3 page interactive dashboard

## Key Findings
- West region is most profitable (14.9% margin)
- Tables sub-category has negative profit margin
- High discounts (40%+) lead to direct losses
- No discount orders have highest profit margin

## Files
- superstore_analysis.sql — SQL queries
- superstore_analysis.xlsx — Excel dashboard
- Superstore_Dashboard.pbix — Power BI file
